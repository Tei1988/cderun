# コンテナコマンド実行

## 概要

`cderun`はエフェメラルコンテナでコマンドを実行し、開発ツールのローカルインストールを必要とせずにクリーンで再現可能な環境を提供します。

## コア機能

### コマンドラッピング
- 任意のコマンドをコンテナ内で実行するようにラップ
- 実行後にコンテナを自動削除 (`--rm`)
- コマンドの終了コードと出力を保持

### 作業ディレクトリマウント
- 現在の作業ディレクトリをコンテナにマウント
- ファイルアクセスと変更機能を維持
- 相対パス操作を保持

### インタラクティブサポート
- インタラクティブコマンド用のTTY割り当て (`--tty`)
- インタラクティブ入力用のSTDIN転送 (`--interactive`)
- フルターミナル体験の保持

## 使用例

### 基本コマンド実行
```bash
cderun node --version
# Node.jsコンテナで'node --version'を実行
```

### インタラクティブシェル
```bash
cderun --tty --interactive bash
# インタラクティブbashシェルを開く
```

### ファイル操作
```bash
cderun python script.py
# 現在のディレクトリにアクセスしてPythonスクリプトを実行
```

## メリット

- **ゼロローカル依存**: ツールをローカルにインストールする必要がない
- **一貫した環境**: 異なるマシン間で同じ動作
- **分離**: コマンドは分離されたコンテナで実行
- **クリーンアップ**: 実行後のコンテナ自動削除
- **透明性**: ネイティブコマンド実行のように動作
